```js

}
alert(numeroDobro(2));
```

4) Criar uma função que recebe três números como parâmetros e retorna a média deles.

```js
function media(numero1, numero2, numero3) {
    return media = (numero1 + numero2 + numero3) / 3
}
alert(parseInt(media(10, 10, 10)));
```
